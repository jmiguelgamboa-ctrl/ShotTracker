# Shot Tracker v0.1.0

A simple one-handed golf shot tracker for GitHub Pages and iPhone Home Screen installation.

## Included
- Saved courses and custom 9-, 10-, or 18-hole courses
- Set par for every hole
- Scorecard-style round view
- Club then result shot logging
- Fairway, Rough, Bunker, Green, Water, OB, and Holed results
- Shot timeline, edit, and undo
- Automatic scoring versus par
- Persistent local storage
- Completed-round history
- PWA and offline shell

## Deploy
1. Create a public GitHub repository such as `ShotTracker`.
2. Upload every file in this folder to the repository root.
3. Commit the upload.
4. Open Settings → Pages.
5. Choose Deploy from a branch, `main`, and `/(root)`.
6. Open the published URL in Safari and use Share → Add to Home Screen.

## Version 0.1 limitations
- Data is local to the current browser/device.
- Shot distance and miss direction are not tracked.
- Bag customization is planned later.
- Shot editing uses a basic prompt in this model.
